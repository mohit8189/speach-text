# Usage Guide

This guide explains how to use the Speech Text App's features.

## Getting Started

When you launch the app, you'll see the **Home Screen** with a button to open the Speech Screen.

## Home Screen

The Home Screen provides:
- App title and description
- A button to navigate to the Speech Recognition screen

**To navigate to the Speech Screen:**
1. Tap the "Open Speech Screen" button

## Speech Screen

The Speech Screen is where all the magic happens. It contains:
1. Language selector
2. Speech recognition button
3. Text input field
4. Text-to-speech button

### 1. Selecting a Language

At the top of the screen, you'll see three language buttons:
- **English**
- **Hindi**
- **Marathi**

**To select a language:**
1. Tap one of the language buttons
2. The selected language will be highlighted in blue
3. Both speech recognition and text-to-speech will use the selected language

## Common Use Cases

### Use Case 1: Voice Notes
1. Select your language
2. Tap speech recognition
3. Speak your note
4. Edit any errors in the text
5. Save or copy the text elsewhere

### Use Case 2: Language Practice
1. Select the language you're learning
2. Type or paste text in that language
3. Use text-to-speech to hear correct pronunciation
4. Practice speaking and use speech recognition to verify

### Use Case 3: Dictation
1. Select your language
2. Use speech recognition to dictate
3. Edit the recognized text
4. Use text-to-speech to verify accuracy

### Use Case 4: Accessibility
1. Use speech recognition for hands-free text input
2. Use text-to-speech to have text read aloud
3. Switch between languages as needed

## Permissions

### First-Time Use

When you first use speech recognition, the app will request permissions:

**On Android:**
- A dialog will appear asking for microphone access
- Tap "Allow" to grant permission
- You can manage permissions in your device settings later

**On iOS:**
- You'll see two permission dialogs:
  1. Microphone access
  2. Speech recognition
- Tap "OK" or "Allow" for both
- You can manage permissions in iOS Settings > Privacy

### If Permissions Are Denied

If you deny permissions, the app will show an error message. To enable them:

**On Android:**
1. Go to Settings > Apps > SpeechTextApp
2. Tap Permissions
3. Enable Microphone

**On iOS:**
1. Go to Settings > Privacy & Security > Microphone
2. Enable for SpeechTextApp
3. Go to Settings > Privacy & Security > Speech Recognition
4. Enable for SpeechTextApp

## Troubleshooting

### Speech Recognition Not Working

**Problem:** No text appears when speaking
- **Solution 1:** Check microphone permissions
- **Solution 2:** Ensure you have an internet connection
- **Solution 3:** Speak more clearly or try again
- **Solution 4:** Check if the correct language is selected

**Problem:** Wrong text is recognized
- **Solution 1:** Speak more slowly and clearly
- **Solution 2:** Reduce background noise
- **Solution 3:** Manually edit the recognized text

### Text-to-Speech Not Working

**Problem:** No audio plays when tapping the TTS button
- **Solution 1:** Check device volume
- **Solution 2:** Ensure the text field is not empty
- **Solution 3:** Try selecting a different language
- **Solution 4:** Restart the app

**Problem:** Wrong pronunciation
- **Solution 1:** Check if the correct language is selected
- **Solution 2:** Some words may not be in the TTS vocabulary

### App Crashes or Freezes

- **Solution 1:** Restart the app
- **Solution 2:** Clear app cache and data
- **Solution 3:** Reinstall the app
- **Solution 4:** Check for app updates

## Tips for Best Results

1. **For Speech Recognition:**
   - Use in a quiet environment
   - Speak at a normal pace
   - Keep sentences clear and simple
   - Pause between thoughts

2. **For Text-to-Speech:**
   - Keep text well-formatted
   - Use proper punctuation for natural pauses
   - Break up very long text into smaller chunks

3. **For Multi-language Support:**
   - Select the correct language before speaking
   - Avoid mixing languages in a single utterance
   - Switch languages as needed for different text

## Advanced Features

### Editing During Speech Recognition

You can edit the text field while speech recognition is active, but new recognized text will replace the current content.

### Using Different Languages

You can:
1. Recognize speech in one language
2. Switch to another language
3. Use text-to-speech in the new language

This allows for translation practice or multi-language note-taking.

## Privacy Note

- No data is stored permanently by this app
- Check your device's privacy settings for more control
